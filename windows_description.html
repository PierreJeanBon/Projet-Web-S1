Présentation de Windows

1985-1992 1992-1995 1995-1999 1999-2001 2001-2005 2005-2007 2009-2010 2012-2013 2013 jusqu'à nos jours

Qu'est ce que Windows ?

Windows est est au départ une interface graphique unifiée produite par Microsoft, qui est devenue ensuite une gamme de systèmes d’exploitation à part entière, principalement destinés aux ordinateurs compatibles PC. Aujourd'hui, le milliardaire Bill Gates est le PDG de la société Microsoft.

1985-1992

1985
Windows 1.00 permet de quitter l’écriture blanche sur fond noir des lignes de commandes MS-DOS pour découvrir une interface noir-blanc dotée d’une barre des tâches. La version 1.01 apporte la couleur et pour la première fois on parle de pilotes de périphériques.
1987
Windows 2.00 n’est pas une révolution, on se souvient simplement que les fenêtres peuvent se chevaucher à l'écran.
1990
Windows 3.00 gère maintenant 16 couleurs et le solitaire fait son apparition.
1992
Windows 3.10, la version classique qui supporte désormais les cartes son, et Windows 3.11, la version pour Workgroups (réseau) naissent successivement cette même année.

1992-1995

1992
Windows NT 3.1 est une évolution de la version Pro de Windows. On parle TCP/IP pour la première fois.
1995
Windows 95 (ou Windows Chicago) est une véritable révolution dans le monde de Windows et du PC. L’interface, dont on dit que le bureau est interactif, est totalement remaniée et fonctionne en mode 32 bits. C’est l’avènement du mode dit "multitâche". C’est aussi l’apparition d’Internet dans l’environnement Windows.

1995-1999

1996
Windows NT 4.0, cette version professionnelle, élaborée partiellement à partir de Windows 95, entre autres pour l'interface, existe en variantes serveur et station de travail.
1998
Windows 98 (ou Windows Memphis) intègre Internet Explorer 4 en standard, la gestion de l'USB et de la FAT 32. Cette version plus stable remplace aisément sa précédente dans le cœur des utilisateurs.

1999-2001

2000
Windows Millenium Edition (ou Windows Me) dont Internet Explorer 5.5 et Windows Media Player 7 semblent être les seules nouveautés, offre pour la première fois un système de restauration du PC à une date antérieure, mais Millenium est surtout le signe de la fin de la série 9x des OS de Microsoft. Les utilisateurs ne verront en lui qu'une sorte de gros patch ou "add-on" pour Windows 98 SE.
2001
Windows XP, est une nouvelle révolution du monde PC, après celle de Windows 95. Le système est enfin stable, les écrans bleus se font de plus en plus discrets. XP (pour les intimes) se décline en deux versions : Home Edition (Edition familiale) et Professional (Professionnel).

2001-2005

2003
Windows Server 2003, comme son nom l'indique, vient prendre la place de Windows 2000 Server et bénéficie des avancées de Windows XP.

2005-2007

2006
Windows Vista (anciennement appelé Longhorn) est mis à disposition du public dans sa version Bêta 2 au mois de juin 2006.
2007
Windows Vista est lancé dans sa version définitive en janvier 2007. Microsoft pense enterrer son ancêtre XP en quelques mois, mais il n’en est rien, car l'arrivée des netbooks relance l’engouement pour XP et Microsoft ne cesse de lui accorder un sursis.

2009-2010

2009
Windows Seven ou Windows 7 prend la succession de Vista en octobre 2009.
2010
Début 2010, les ventes de Windows 7 semblent avoir démarré en trombe, on parle de 90 millions de copies vendues en trois mois.

2012-2013

2012
Windows 8 est sorti le 26 octobre 2012. Avec ses tuiles, il ne fait pas l'unanimité, d'autant plus que le célèbre "Menu démarrer" a purement et simplement disparu.
2013
Windows 8.1 a réintégré le "menu démarrer" qui a manqué à des millions d'utilisateurs... dès le 17 octobre 2013.

2013 jusqu'à nos jours

2015
Windows 10 est disponible depuis le 29 juillet 2015, sous forme de mise à jour gratuite (KB3035583), si l'on est possesseur d'une version de Windows 7, 8 ou 8.1
2017
Il semble que la rumeur avait raison et qu'il n'y aura pas de Windows 11... après les versions, 1507 (Threshold 1), 1511 (Threshold 2 - November Update), 1607 (Redstone 1 - Anniversary Update), 1703 (Redstone 2 - Creators Update), voici, depuis le 18 octobre 2017 la version 1709 (Redstone 3 - Fall Creators Update) et nous serions dans l'attente de la version 1803 (Redstone 4) pour début 2018.
2018
Depuis le 30 avril 2018, la mise à jour majeure de Windows 10, la version 1803 (Redstone 4 - April 2018 update) est disponible pour tous les utilisateurs...
2019
Le 6 juillet 2019, nouvelle mise à jour de Windows 10, version 1903, 18362.207.

Et pourquoi pas Windows 9 ?

Lors de la présentation officielle de Windows 10, il n’a pas été donné de véritable explication concernant le passage de Windows 8 à Windows 10. L'orateur s’est contenté d'insister sur le fait que l'approche du nouveau produit est fondamentalement différente de ce qui a existé jusqu'à ce jour et qu'il fallait le souligner, ce que n’aurait pas permis un nom tel que Windows 9. Selon lui, il n’était malheureusement pas possible de l'appeler Windows One, en référence aux produits récents comme OneNote, XBox One ou encore OneDrive.

Une autre explication circulant sur le Web prétend que la version succédant à Windows 8 porte le patronyme de Windows 10, et non celui de Windows 9, pour des raisons de programmation. Quelques programmeurs auraient créé un code pour identifier la version de Windows avant l’installation de leur logiciel ; code qui dit "if(version.StartsWith("Windows 9")) { /* 95 and 98 */ } else {" ou, en langage décodé, "si la version de Windows sur ce PC commence par "Windows 9", il s’agit de Windows 95 ou 98 et le logiciels ne pourra pas être installé" rendant ainsi la nouvelle version de Windows obsolète… Selon cette explication, ce serait par magnanimité que Microsoft a choisi d'ignorer le 9 au profit du 10.
